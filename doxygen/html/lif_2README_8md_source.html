143 ## LIF command help
144  * lif add lifimage lifname from_ascii_file
145  * Add ASCII file converted to E010 format to existing LIF image on SD card
146  * lif add /amigo1.lif TEST1 /test.bas
147  * lif add /amigo1.lif TREK85 /TREK85/TREK85.BAS
148  * Notes:
149  * Strings must be no longer then sector size - 3
150  * Any trailing "\n" and/or "\r" are coverted to "\n" when stored in LIF file
151 
152  * lif addbin lifimage lifname from_lif_image
153  * Adding a binary LIF image files to another LIF image
154  * Examples
155  * lif addbin /amigo1.lif TEST /test.lif
156  * lif addbin /amigo1.lif TREK85 /TREK85/trek.lif
157  * Notes about TREK85 in the examples
158  * Author: TREK85 port was done by Martin Hepperle
159  * https://groups.io/g/hpseries80/topic/star_trek_game_for_hp_85/4845241
160 
161  * lif create lifimage liflabel directory_sectors total_sectors
162  * Create a new LIF image
163  * Example
164  * lif create /amigo3.lif AMIGO3 15 1120
165  * This formats an LIF image file with 15 directory sectors and a length of 1120 (16 * 2 * 35) sectors
166 
167  * lif del lifimage lifname
168  * Delete a file from LIF image on SD card
169  * Example
170  *lif del /amigo1.lif TREK85
171 
172  * lif dir lifimage
173  * Directory listing of LIF image
174  * Example:
175  <pre>
176  lif dir amigo1.lif
177  Volume:[AMIGO2] Date:[<EMPTY>]
178  NAME TYPE START SECTOR SIZE RECSIZE DATE

182  GPIB-T e020h 15h 1536 256 <EMPTY>
183  GPIB7 e020h 1bh 256 256 <EMPTY>
184  AMIGO2 e020h 1ch 256 256 <EMPTY>
185  TREK85 e010h 1dh 27615 256 Tue Jun 27 16:28:28 2017
186  HELLO3 e010h 8bh 344 256 Sun Jun 18 18:13:24 2017
187  TREK85B e010h 8dh 27615 256 Tue Jun 27 16:28:28 2017
188 
189  9 Files

191  231 Used sectors
192  873 Free sectors
193  </pre>
194 
195  * lif extract lifimage lifname to_ascii_file
196  * Extracts E010 file from LIF image converting to ASCII file on SD card
197  * Example
198  lif extract /amigo1.lif HELLO3 /HELLO3.BAS
199 
200  * lif extractbin lifimage llifname to_lif_image
201  * Extracts LIF file from a LIF image and saves it as new LIF image
202  * Example
203  * lif extractbin /amigo1.lif HELLO3 /hello3.lif
204 
205  * lif rename lifimage oldlifname newlifname
206  * Renames file in LIF image
207  * Example
208  * lif rename /amigo1.lif HELLO3 HELLO4
