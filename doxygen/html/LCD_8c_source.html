50  * ASCII / DEC / HEX

52  * Ctrl+c / 3 / 0x03 - Change width to 20

54  * Ctrl+e / 5 / 0x05 - Change lines to 4
55  * Ctrl+f / 6 / 0x06 - Change lines to 2

57  * Ctrl+h / 8 / 0x08 - Software reset of the system

62  * Ctrl+m / 13 / 0x0D - Change baud to 9600bps
63  * Ctrl+n / 14 / 0x0E - Change baud to 14400bps
64  * Ctrl+o / 15 / 0x0F - Change baud to 19200bps
65  * Ctrl+p / 16 / 0x10 - Change baud to 38400bps
66  * Ctrl+q / 17 / 0x11 - Change baud to 57600bps
67  * Ctrl+r / 18 / 0x12 - Change baud to 115200bps
68  * Ctrl+s / 19 / 0x13 - Change baud to 230400bps
69  * Ctrl+t / 20 / 0x14 - Change baud to 460800bps
70  * Ctrl+u / 21 / 0x15 - Change baud to 921600bps
71  * Ctrl+v / 22 / 0x16 - Change baud to 1000000bps
72  * Ctrl+w / 23 / 0x17 - Change baud to 1200bps
73  * Ctrl+x / 24 / 0x18 - Change the contrast. Follow Ctrl+x with number 0 to 255. 120 is default.
74  * Ctrl+y / 25 / 0x19 - Change the TWI address. Follow Ctrl+x with number 0 to 255. 114 (0x72) is default.
75  * Ctrl+z / 26 / 0x1A - Enable/disable ignore RX pin on startup (ignore emergency reset)
76  * '+' / 43 / 0x2B - Set RGB backlight with three following bytes, 0-255
77  * ',' / 44 / 0x2C - Display current firmware version
78  * '-' / 45 / 0x2D - Clear display. Move cursor to home position.
79  * '.' / 46 / 0x2E - Enable system messages (ie, display 'Contrast: 5' when changed)
80  * '/' / 47 / 0x2F - Disable system messages (ie, don't display 'Contrast: 5' when changed)
81  * '0' / 48 / 0x30 - Enable splash screen
82  * '1' / 49 / 0x31 - Disable splash screen
