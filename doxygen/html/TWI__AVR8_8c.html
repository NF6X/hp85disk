Detailed Description

TWI Peripheral Driver (AVR8)

On-chip TWI driver for the 8-bit AVR microcontrollers.

Note
This file should not be included directly. It is automatically include d as needed by the TWI driver dispatch header located in LUFA/Drivers/Peripheral/TWI.h.
