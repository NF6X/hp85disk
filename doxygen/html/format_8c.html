Disk Layout https://groups.io/g/hpseries80/wiki/HP-85-Program-Control-Block-(BASIC-header),-Tape-directory-layout,-Disk-directory-layout.

DISK LAYOUT The HP-85 disks used the LIF (Logical Interchange Format) disk layout. The first 2 sectors on the disk (cylinder 0, head 0, sector 0-1) contained the VOLUME sectors. The important things in the VOLUME SECTORS were thus:

BYTES DESCRIPTION

0-1 LIF identifier, must be 0x80, 0x00 (0x8000) 2-7 6-character volume LABEL 8-11 directory start block (always 0,0,0,2 = 0x00000002) 12-13 LIF identifer for System 3000 machines (always 0x10,0 = 0x1000) 14-15 always 0 16-19 # of sectors in DIRECTORY (usually 0,0,0,something) 20-21 LIF version number (always 0,1 = 0x0001) 22-23 always 0 24-27 number of tracks per surface 28-31 number of surfaces 32-35 number of sectors per track 36-41 date and time that the volume was initialized (YY,MM,DD,HH,mm,SS) All date values are in BCD format. YY is (year-1900). HH is 0-23.LIF data structures used by stand alone formatting utility LIF Volume label structure
