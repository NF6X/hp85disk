127 ## AVR ATMEGA1284P pin assignments for HP85 Disk
128  * GPIB: Each GPIB pin (8 data and 8 control lines ) attach to CPU with a 120 ohm current limit resistor .
129  * Each GPIB pin (8 data and 8 control lines ) have a 10K pull-up resistor to VCC.
130  * ISP header: MOSI,MISO,SCK,/Reset connects directly to ISP header
131  * Micro SD Interface: MOSI,MISO,SCK attach to CPU function via a 1k series resistor.
132  * Micro SD interface has level shifters and internal 5V to 3.3V regulator
133  * RS232 TTL: connect to FTDI232 USB board which also provides 5V VCC power to all circuits..
134  * I2C: SCL,SDA connect to optional DS1307 RTC board with each line having a 2k2 pull-up

169 ## Parallel Poll Response circuit
170  * Uses: Three chips 74HC05, 74HC32, 74HC595
171  * Parallel Poll Response must be less then 2 Microseconds therefore we use hardware to do it!
172 
173 
174 <pre>
175  ATMEGA HC595 HC05
176  +----V----+ +-V-+
177  3 PB3 -------- 12 |RCLK Q0| 15 -x- 1 | | 2 --- GPIB D8
178  6 MOSI ------- 14 |SER Q1| 1 -x- 3 | | 4 --- GPIB D7
179  8 SCK -------- 11 |SRCLK Q2| 2 -x- 5 | | 6 --- GPIB D6
180  9 IFC -------- 10 |SRCLR Q3| 3 -x- 9 | | 8 --- GPIB D5
181  HC32 | | | | | 7 GND
182  +-V-+ | | | | |14 VCC
183  EOI 2| | | | | +---+
184  ATN 1| | | | \--- each line has its own
185  | | 1--13 |/OE | 10K resistor to GND
186  VCC 14| | | | 16 VCC
187  GND 7| | | | 8 GND
188  +---+ +---------+
189 </pre>
190 
191 
192 Notes: When both EOI and ATN are low the HC32 enables HC595 outputs
193  * If any HC595 output is high the GPIB bus bit will be pulled low
194  * IFC low resets the HC595 outputs low - so the HC05 outputs will float.
