#define AVR_DIR_IN (   a)    BIT_CLR( a ## _DDR, a ## _BIT)

Notes about AVR and PIC port BIT differences. AVR PIC DDR 1=out, 0 =in TRIS 0=out,1=in PORT=val same as LATCH=val PORT=val same as LATCH=val val=PORT, reads LATCH val=PORT reads PIN state val=PIN, reads PIN state val=LATCH reads latch Notes: "SENSOR" gets convereted into the required DDR and BIT definitions. We use the comma operator to return the PIN test. We do not use {} around the statements so they behave like functions. Consider what would happen if you used breaces (should be obviious) if ( GPIB_IO_RD(SENSOR) ) { printf"SENSOR"); } else { do stuff }
