lifdir_t *  DIR 
)

Convert a user ASCII file into HP85 0xE010 LIF format We must know the convered file size BEFORE writting So we must call this function TWICE 1) find out how big the converted file is (DIR is NULL) 2) Specify where to put it (DIR is set)

Parameters
[in]userfileUser ASCII file source
[in]*DIRWhere to write file if set (not NULL)
Returns
size of LIF image in bytes, or -1 on error

Read and process into image file

HP85 ASCII LIF records have a 3 byte header Byte 1: (0xdf = data) or (0xef = EOF) Byte 2: LSB number of bytes to next LIF record Byte 3: MSB nunber of bytes to next LIF record EOF has these 4 bytes 0xdf 0x00 0x00 0xef 0xff 0xff Example small file: 0xdf 0x05 0x00 "HP85\r" 0xdf 0x00 0x00 0xef 0xff 0xff ....
