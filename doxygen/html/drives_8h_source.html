138 /*
139  When packed this is 19 bytes
140  UNIT DESCRIPTION
141  U1 Generic Unit Type, 0 = fixed, 1 - floppy, 2 = tape
142  OR with 128 implies dumb can not detect media change
143  U2-U4 Device number in BCD
144  XX XX XY, X = BCD Unit number, Y = option number
145  U5-U6 Number of bytes per block, MSB-LSB

149 U11-U12 Continuous average transfer rate for long transfers kB/s
150 U13-U14 Optimal retry time in 1O's of milliseconds
151 U15-U16 Access time parameter in 1O's of milliseconds
