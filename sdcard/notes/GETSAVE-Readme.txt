The GETSAV program will let you GET or SAVE store ASCII files on the HP85

Example: To LOAD a BASIC program as TEXT
LOADBIN "GETSAV"
GET "TEST_T"

Example: To SAVE a BASIC program as TEXT
SAVE "TEST_T"
